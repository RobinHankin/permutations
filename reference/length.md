# Various vector-like utilities for permutation objects.

Various vector-like utilities for permutation objects such as `length`,
`names()`, etc

## Usage

``` r
# S3 method for class 'word'
length(x)
# S3 method for class 'permutation'
length(x) <- value
# S3 method for class 'word'
names(x)
# S3 method for class 'word'
names(x) <- value
```

## Arguments

- x:

  permutation object

- value:

  In function `names<-.word()`, the new names

## Details

These functions have methods only for word objects; cycle objects use
the methods for lists. It is easy to confuse the *length* of a
permutation with its `size`.

It is not currently possible to set the length of a permutation; this is
more trouble than it is worth.
